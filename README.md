Projeto Fullstack FemTech-blog
=======================

Backend: Node + Express + Prisma + SQLite
Frontend: Next.js

Instruções rápidas:

1) Backend
$ cd backend
$ npm install
$ cp .env.example .env
$ npx prisma migrate dev --name init
$ npm run dev

2) Frontend
$ cd frontend
$ npm install
$ npm run dev

Frontend em http://localhost:3000
Backend em  http://localhost:4000
